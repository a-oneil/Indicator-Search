Microsoft Security Response Center

Report abuse to Microsoft Security Response Center (MSRC) for activity originating from a Microsoft online service, such as Microsoft Azure, Bing, OneDrive, and Office365. For the common types of abuse reported to MSRC, please review the "Reportable Incident Types" table below.










Incident Type Threat Type Description
Account Compromise/Unauthorized Access IP Address Relates to when an attacker has successfully compromised an account.
Brute Force IP Address Relates to when an attacker uses trial-and-error to exhaustively explore all possible values to unlock an asset (Password Spray, Dictionary Attack).
Denial of Service IP Address Relates to an interruption or suspension of services or hosts through a large number of interactions with the target or by depleting resources to the point where the target's functionality is affected (Flooding, Reflection Attack, Resource Depletion).
Malware IP Address, URL Relates to the hosting or distribution of malicious software that is specifically designed to disrupt, damage, or gain unauthorized access to a computer system.
Responsible AI URL Relates to Microsoft's Responsible AI principles: reliability and safety, fairness, privacy and security, inclusiveness, transparency and accountability. Only available through the Report Abuse Portal.
Scanning/Scraping IP Address Relates to when an attacker attempts to determine the state of the ports, collect information about the OS, or extract data from human-readable output from exposed sources.
Phishing URL Relates to when an attacker creates an invented scenario or otherwise engages an individual to solicit information or manipulate the target into performing actions that serve attackers' interests (E.g. Phishing).
Spam IP Address Relates to any unsolicited emails.
